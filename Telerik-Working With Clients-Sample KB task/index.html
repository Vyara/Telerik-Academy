How to implement a background changing functionality.

Problem:

You want to have an easy access and easy-to-use option for setting a background color to your page.

Description:

Many users enjoy websites, which give them the options for content personalization.
Color themes are one of the most noticeable and enjoyed features, which could give your web page that desired personal touch.
A basic but powerful demonstration of this, could be implementing a simple form containing 5 radio buttons,
representing 5 different colors. When a specific radio button is clicked, the background color of its
container(could be a form or a page) is set to the color,
which the selected button represents.

Solution:

You can create this simple form by following the below steps:

  1. Include the latest version of jQuery in your html project. You can use the CDN option.
  2. Create an html form element and give it a specific id value(optionally, you can set a css style of your choice): 
    <form action="" id="some-id" style="width: 500px"></form>
  3. In the form, create 5 input elements with type="radio" attributes and name="_{name of your choice}" 
    <input type="radio" name="_background-color" value="red">Red
    Note: You can leave the radio button representing your page's default background-color checked by default: 
      <input type="radio" name="_background-color" value="white" checked="checked">White
  4. In a script tag, include the javascript functionality, using jQuery selectors as follows:
    1. Select the form tag by its id: 
      $('#the-form')
    2. Attach a onclick event onto it, based on the inputs' common name attribute: 
      $('#the-form').on('click', 'input[name=_background-color]', function () {});
    3. Within the function, change the form tag's background color by using the css method and apply the color of the targeted(selected) checkbox(the this object): 
       $('#the-form').css("background-color", this.value);

    The whole script code should look something like this: 

    $('#the-form').on('click', 'input[name=_background-color]', function () {
        $('#the-form').css("background-color", this.value);
        });

Eventually, you should have a sample functionality like the example below:

White
Red
Green
Blue
Yellow

And that's it! Now you can implement this option in your project and give the users the option to personalize the site with a color of their choice!
