01. Write down in a text file all the major similarities and differences you can find between ASP.NET Web Forms and ASP.NET MVC

Similarities
They both share API's related to http and user authentication.
Have the same startup events in Global.asax.
Both can be developed through the same IDE (Visual Studio)
and use the same NuGet packages for development.

Differences
Webforms relies heavily on controls, postback(event driven) and has less control over HTML.
MVC provides more control over the HTML output and it looks a lot cleaner.
MVC uses Controllers, WebForms uses Code Behind
They use different types View Engines.
MVC imposes better separation of concerns.
